What is this?

Ever wanted to put an interesting quote on your website?
Well, this app solves just this problem and nothing else :)
You create a list of your favorite quotes and get an html snippet to embed into your page

How it works?

1. Create a list of quotes

Add your favorite quotes to the list.
You can add tags to the quotes to filter them later.

2. Create a widget

Widget configures how the quote will look like on your site.
Widget displays random quote from the list - you can have predefined list of quotes or just display random quote for certain tags.

3. Embed the stuff

After widget is configured you can use html code to put it in your site - it comes in two flavors - iFrame and png image. For iFrame you are in total control of how it's going to look like - css is editable in widget config.
